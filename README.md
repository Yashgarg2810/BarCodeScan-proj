# BarCodeScan

A Flutter application for SMS functionality.

## Getting Started

This project is a Flutter application. To get started, you need to:

1. Install Flutter SDK
2. Set up your development environment
3. Run the application

### Prerequisites

- Flutter SDK (3.0.0 or higher)
- Dart SDK
- Android Studio / VS Code
- Android SDK (for Android development)
- Xcode (for iOS development, macOS only)

### Installation

1. Download Flutter SDK from [flutter.dev](https://flutter.dev/docs/get-started/install)
2. Extract it to a location on your computer
3. Add Flutter to your PATH environment variable
4. Run `flutter doctor` to verify the installation

### Running the App

1. Clone this repository
2. Navigate to the project directory
3. Run `flutter pub get` to install dependencies
4. Run `flutter run` to start the app

### Project Structure

```
lib/
  ├── main.dart          # Main application entry point
  └── ...
android/                  # Android-specific files
ios/                     # iOS-specific files
web/                     # Web-specific files
pubspec.yaml             # Project dependencies
```
